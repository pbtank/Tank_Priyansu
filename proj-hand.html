CAD Integrated with Hand Gestures

  • Associated with: Standalone Project
  • Started: 2022
  • Completed: Ongoing

    • Abstract

    This project is an effort to make use of CAD systems easy. Instead of spending hours with unintuitive mouse to interact with 3D models in CAD software, this project provides an intuitive for this interaction. Various hand gestures are used to manipulate an object for its rotation around various axes. Currently it supports rotation and Home view shortcut. For hand gesture tracking, it uses Mediapipe Hand Gesture library.
